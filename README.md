# Multi Step Form (React + TypeScript)

This is a simple multi-step form project that I built to practice working with React, TypeScript and modern form validation.

The main goal of this project was to understand how to manage form state across multiple steps and build reusable UI components with a clean structure.

## Features
- Multi step form with validation
- React Hook Form + Zod
- Modern UI with Tailwind and shadcn
- Reusable components
- Clean and readable code
- TypeScript for type safety

## Tech Stack
React  
TypeScript  
Tailwind CSS  
shadcn/ui  
React Hook Form  
Zod
